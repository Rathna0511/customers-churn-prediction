# Customer Churn Prediction

## Overview
This project predicts whether a customer will churn using machine learning.

## Features
- Data preprocessing
- Feature engineering
- Model building (Random Forest)
- SHAP explainability

## Tech Stack

- Pandas
- Scikit-learn
- SHAP
